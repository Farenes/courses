# Лабораторная №5. Разработка Telegram-бота

## Тема и цель работы
Тема: разработка Telegram-бота с интеграцией внешнего API.

Цели:
1. Приобрести практические навыки разработки Telegram-ботов на выбранном языке программирования с использованием соответствующих фреймворков.
2. Освоить принципы работы с внешними API (Application Programming Interface): отправка HTTP-запросов, обработка ответов в формате JSON/XML, управление ошибками.
3. Научиться проектировать архитектуру простого бота, обрабатывающего пользовательские команды и интерактивные элементы.
4. Закрепить навыки работы с системой контроля версий (Git) и публикации проекта на хостинге.

## Общая постановка задачи
Необходимо разработать функционального Telegram-бота, который в обязательном порядке взаимодействует с одним или несколькими публичными внешними API. Бот должен выступать в роли удобного интерфейса между пользователем мессенджера и данными, предоставляемыми сторонним сервисом.

Общая логика работы: 
- Пользователь отправляет боту команду или текст. 
- Бот формирует на его основе запрос к определенному внешнему API, получает и обрабатывает данные (фильтрует, форматирует, извлекает ключевую информацию) и возвращает понятный результат пользователю в чат Telegram.

## Требования к боту
Базовый функционал:
- Наличие команд /start и /help с приветственным сообщением и описанием возможностей бота.
- Корректная обработка стандартных ошибок (неверный ввод пользователя, недоступность API, сетевые проблемы) с отправкой информативного сообщения пользователю.

Работа с внешним API:
- Бот должен совершать HTTP-запросы (GET/POST) к публичному API.
- В коде должна быть реализована парсинг ответа API (извлечение данных из JSON/XML).
- Ключи API (если требуются) должны храниться безопасно (в переменных окружения или конфигурационных файлах, исключенных из публичного репозитория).
- Логика обращения к API должна быть вынесена в отдельную функцию/модуль.

Технические требования:
- Чистый, читаемый код с комментариями.
- Проект размещен на GitHub.
- README.md-файл с описанием проекта, инструкцией по установке, запуску и списком используемых технологий.

## Варианты
### 1. Определение слова в словаре
- Команда: /define serendipity — выдает определение слова на английском.
- API: [Free Dictionary API](https://dictionaryapi.dev/).
### 2. Синонимы слова 
- Команда: /synonym большой — выдает список синонимов.
- API: [WordsAPI (часть RapidAPI) (есть бесплатный лимит)](https://www.wordsapi.com/).
### 3. Случайная википедия статья
- Команда: /wiki_random — присылает название и краткую выдержку из случайной статьи.
- API: [Wikipedia API (Action API)](https://www.mediawiki.org/wiki/API:Action_API).
### 4. Поиск книг по автору/названию
- Команда: /books "Достоевский" — выдает список книг.
- API: [Open Library Search API](https://openlibrary.org/dev/docs/api/search).
### 5. Переводчик (упрощенный)
- Команда: /translate en Привет, мир! — переводит текст на указанный язык.
- API: [LibreTranslate (есть публичный сервер)](https://libretranslate.com/docs/).
### 6. QR-код генератор: 
- Команда: /qr https://example.com — генерирует и присылает картинку с QR-кодом.
- API: [GoQR.me API (простой GET-запрос)](https://goqr.me/api/doc/create-qr-code/).
### 7. Сокращатель ссылок: 
- Команда: пользователь отправляет длинную ссылку, бот возвращает короткую (bit.ly, tinyurl).
- API: [T.ly (TinyURL) API](https://tinyurl.com/app/dev).
### 8. Рекомендация случайного фильма
- Команда: /random_movie 2010s comedy — случайный фильм по критериям (год, жанр).
- API: [OMDb API](https://www.omdbapi.com/).
### 9. Случайная шутка
- Команда: /joke — выдает шутку.
- API: [Official Joke API](https://github.com/15Dkatz/official_joke_api?tab=readme-ov-file).
### 10. Синонимы слова 
- Команда: /synonym большой — выдает список синонимов.
- API: [Datamuse API (не требует ключа)](https://www.datamuse.com/api/).
### 11. Переводчик (упрощенный)
- Команда: /translate en Привет, мир! — переводит текст на указанный язык.
- API: [MyMemory API (бесплатно с лимитами)](https://mymemory.translated.net/doc/).
### 12. Случайная интересная картинка кота
- Команда: /pic присылает случайное изображение котика.
- API: [The Cat API](https://thecatapi.com/).
### 13. Случайная интересная картинка собаки
- Команда: /pic присылает случайное изображение собачки.
- API: [The Dog API](https://thedogapi.com/).
### 14. Погода в городе: Классический бот. 
- Команда: /weather Москва или просто отправка геолокации.
- API: [OpenWeatherMap](https://openweathermap.org/current?collection=current_forecast).
### 15. Конвертер валют
- Команда: /exchange 100 USD to EUR.
- API: [ExchangeRate-API (бесплатный тариф с лимитами)](https://www.exchangerate-api.com/docs/overview)
